# image-captioning-project

Before running whole project use `pip install -r requirements.txt`.

Used https://www.kaggle.com/datasets/adityajn105/flickr8k

When running the code keep in mind that the dataset is about 1GB big; depending on your internet connection that might take some time. Consider GitHub Codespaces for a speedier download as it already runs in the cloud, so downloads there tend to be way quicker.

Image preprocessing was made in such a way that it fits ResNet.
